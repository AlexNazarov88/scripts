=== инструменты конвертации xml --> bin ===
* all_3dmodels_to_bin.bat / all_3dmodels_to_bin.py - конвертирует абсолютно все модели (есть хардкод директорий 
с определенными файлами, так что если будуть менятся и добавляться файлы - нужно будет менять код утилиты)

* есть скрипты для конвертации отдельных моделей и предметов персонажей, но с тем же минусом.

* xml-to-bin - универсальный, ему нужно передавать только путь для директории (рекомендуется использовать его)

exlude_anims\exclude.py позволяет удалить ненужные анимаци из xml модели.
Моделька Остина используется сразу на двух проектах, поэтому требуется вырезать ненужные анимации.
Чтобы сконвертировать ostin_skeleton.xml в ostin_skeleton.xml.bin нужно положить его в base_mm/models/Ostin. 
В файле exclude_anims/required_animations нужно указать требуемые анимации, а затем запустить _ostin_skeleton_xmltobin.bat
